# Actividad 2 - Implementación de una página HTML5 básica

## Nombre del proyecto

The Chill | Tu momento Chill

## Propósito de la página

Para esta actividad decidí crear una página web básica sobre The Chill, un emprendimiento enfocado en la venta de granizados, sodas, micheladas y bebidas preparadas.

El propósito de la página es presentar la marca de una forma sencilla, clara y organizada. La intención es que una persona que visite el sitio pueda entender rápidamente qué es The Chill, qué productos ofrece y qué tipo de experiencia busca brindar.

La página está dirigida principalmente a familias, jóvenes y grupos de amigos que desean compartir una bebida refrescante en un espacio cómodo, llamativo y agradable.

## Cómo apliqué HTML5 semántico

En el desarrollo de la página utilicé etiquetas semánticas para que la estructura fuera más clara, tanto para los usuarios como para los navegadores y lectores de pantalla.

Las etiquetas principales utilizadas fueron:

- `<header>`: para ubicar el nombre de la página, una breve descripción y la navegación principal.
- `<nav>`: para agrupar los enlaces internos de navegación.
- `<main>`: para contener el contenido principal de la página.
- `<section>`: para dividir el contenido en bloques temáticos.
- `<article>`: para presentar cada producto como un contenido independiente.
- `<aside>`: para mostrar información complementaria sobre la experiencia de marca.
- `<footer>`: para cerrar la página con información final.

También organicé el contenido con títulos jerárquicos usando `<h1>`, `<h2>` y `<h3>`, lo que permite que la lectura sea más ordenada.

## Cómo apliqué accesibilidad

Para mejorar la accesibilidad de la página tuve en cuenta varias prácticas importantes:

- La imagen principal tiene un atributo `alt` descriptivo.
- El formulario tiene etiquetas `<label>` correctamente asociadas a cada campo mediante los atributos `for` e `id`.
- Se agregó un enlace para saltar directamente al contenido principal, útil para personas que navegan con teclado.
- Los enlaces, campos y botones tienen un estado visible cuando reciben foco.
- Se usó `aria-label` en la navegación y en algunos elementos para dar mayor claridad.
- Los colores seleccionados tienen buen contraste entre texto y fondo.
- El contenido está escrito de forma clara y organizado por secciones.

## Cómo visualizar la página

Para ver la página se debe abrir el archivo `index.html` en cualquier navegador web.

## Validación del código

Antes de entregar la actividad, el archivo `index.html` se valida con la extension W3C Web Validator
para VSCode. Esta validación permite revisar si el código HTML tiene errores de estructura.

## Conclusión

Esta actividad me permitió aplicar una estructura básica de HTML5 usando buenas prácticas de semántica y accesibilidad. Además, me ayudó a entender que una página web no solo debe verse bien, sino que también debe estar organizada correctamente y ser fácil de navegar para diferentes tipos de usuarios.
